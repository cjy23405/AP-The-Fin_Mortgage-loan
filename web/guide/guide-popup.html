Attribute

Layer

Name Type Default Description
data-layer string 해당 객체를 레이어 팝업으로 지정 및 레이어 팝업 이름 지정
data-layer-display string block 레이어 팝업 최상위 부모의 display 스타일 속성 변경시 이용
data-layer-timer-close number 레이어가 자동으로 닫히는 시간을 지정 1000 = 1초
data-focus boolean true 레이어가 열릴 때 해당 레이어로 포커스 가게 하는 기능 활성 여부
data-scroll-block boolean true 레이어가 열릴 때 페이지 스크롤이 막히는 기능 활성 여부
data-cycle-focus boolean true 레이어 내 포커스 순환 기능 활성 여부
data-dim-close boolean true 딤 영역 클릭시 닫힘 기능 활성 여부
data-opener-focus boolean true 레이어 닫힌 후 오프너로 포커스 돌아가는 기능 활성 여부

Opener

Name Type Default Description
data-layer-open string 해당 객체를 레이어 팝업 오프너로 지정
해당하는 레이어 팝업 이름을 값으로 넣어준다
data-toggle boolean false 토글 기능 활성 여부

Closer

Name Type Default Description
data-role string 'layerClose' 값을 넣어 레이어 닫기 버튼으로 지정
해당 버튼 부모 레이어의 닫기 버튼으로 동작한다

Events

Name Arguments Description
layerBeforeOpened (event) 레이어가 열리기 직전에 발생
layerOpened (event) 레이어가 열릴 때 발생
layerAfterOpened (event) 레이어가 열리는 애니메이션이 끝난 후 발생
layerBeforeClosed (event) 레이어가 닫히기 직전에 발생
layerClosed (event) 레이어가 닫힐 때 발생
layerAfterClosed (event) 레이어가 닫히는 애니메이션이 끝난 후 발생
